Busque Ajuda

“No Brasil, acontece uma morte por suicídio a cada 45 minutos, mas para cada morte temos outras 20 tentativas. Os números são altos e preocupantes”, explicou O psiquiatra Humberto Müller. Os casos são muitos, por isso, vimos uma necessidade de fazer essa campanha, sabemos que é um assunto delicado, mas precisamos falar sobre. Afinal, não são poucos que sofrem com essa maldita doença.

Esse momento Não é fácil, sabemos disso, mas lembre-se, você não está sozinho, essa fase irá passar. Existem diversos meios para encontrar ajuda, o CVV, psicólogos, até mesmo de hospitais públicos, mas nossa recomendação é um site chamado Nova-mente, nele é possível agendar consultas com psicológos online, e tudo isso gratuitamente.
